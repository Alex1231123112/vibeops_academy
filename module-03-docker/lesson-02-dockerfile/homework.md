# Домашнее задание: Dockerfile

## Задание: Dockerfile для своего проекта

1) Создайте Dockerfile для вашего Python-проекта (можно взять основу из `examples/Dockerfile.simple`).  
2) Соберите образ: `docker build -t my-app .`  
3) Запустите: `docker run --rm my-app`  

## Дополнительно (желательно)

- Добавьте `EXPOSE 8000` и запуск веб-приложения (если у вас FastAPI/Flask).
- Добавьте `.dockerignore` (минимум: `.git`, `.env`, `__pycache__`, `.venv`).

## Критерии сдачи

- [ ] Образ собирается без ошибок
- [ ] Контейнер запускается и выполняет программу
- [ ] Есть инструкция запуска (2–5 строк) в README репозитория
