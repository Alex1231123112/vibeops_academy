# Домашнее задание: Урок 3.1 (Что такое Docker)

## Задание: собрать и запустить “Hello container”

1) Создайте папку `hello-docker/` и файл `main.py`:

```python
print("Hello from container!")
```

2) Создайте `requirements.txt` (может быть пустым).

3) Создайте `Dockerfile` (можно взять за основу пример из урока 3.2):
- базовый образ `python:3.12-slim`
- копирование файлов
- запуск `python main.py`

4) Соберите образ:
- `docker build -t hello-docker .`

5) Запустите контейнер:
- `docker run --rm hello-docker`

## Критерии сдачи

- [ ] Образ собирается без ошибок
- [ ] Контейнер запускается и печатает “Hello from container!”
- [ ] Вы можете объяснить разницу “образ” vs “контейнер” в 2–3 предложениях
