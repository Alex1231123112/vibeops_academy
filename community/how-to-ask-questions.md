# Как задавать вопросы

## Хороший вопрос содержит

1. **Контекст** — что вы делаете, какой урок
2. **Код** — минимальный пример, который воспроизводит проблему
3. **Ошибка** — полный текст ошибки (traceback)
4. **Что пробовали** — какие шаги уже делали

## Пример

```
Урок 1.1, задание с калькулятором.
При делении на ноль программа падает.

Код:
[вставка кода]

Ошибка:
ZeroDivisionError: division by zero

Пробовал добавить if b != 0, но не знаю куда.
```
