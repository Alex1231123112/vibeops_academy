# Часто задаваемые вопросы

## Общие

**Q: Нужен ли опыт программирования?**  
A: Базовые знания помогут, но курс рассчитан на новичков. Главное — желание учиться.

**Q: Платный ли курс?**  
A: Нет, все материалы бесплатны. GitHub и Telegram — открытые.

**Q: Сколько времени занимает?**  
A: Около 5–10 часов в неделю при структурированном прохождении.

## Технические

**Q: Какой Python нужен?**  
A: Python 3.11+ (рекомендуется 3.12).

**Q: Windows или Mac?**  
A: Оба подходят. Команды терминала могут отличаться (PowerShell vs Bash).

**Q: Нужен ли платный GitHub?**  
A: Нет, бесплатного аккаунта достаточно.
