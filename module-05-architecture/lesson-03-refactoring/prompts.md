# Промпты для AI: Урок 5.3 (Рефакторинг)

База (канонично в библиотеке):

- Шаблон: [prompts-library/template.md](../../prompts-library/template.md)
- Архитектура/рефакторинг: [prompts-library/for-architecture.md](../../prompts-library/for-architecture.md)

Ниже — 1–2 промпта, специфичные для урока.

## 1) Рефакторинг “не меняй поведение” + тест-план

```
Вот код:
[вставь]

Сделай рефакторинг.
Требования:
- не меняй поведение
- маленькими шагами (2–4 шага)
- после каждого шага скажи, как проверить, что всё ок
Ответ: патч + чек-лист проверки.
```

## 2) Подготовить PR-описание

```
Я сделал рефакторинг.
Вот список изменений:
[вставь]

Составь описание для PR:
- что и зачем
- как проверял (test plan)
- риски/что могло сломаться
```
