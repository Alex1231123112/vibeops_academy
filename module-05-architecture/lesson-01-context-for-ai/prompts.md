# Промпты для AI: Урок 5.1 (Контекст)

## 1) “Какие файлы нужны для контекста?”

```
Я хочу, чтобы ты помог с задачей: [задача].
Какие 5–10 файлов/папок мне показать, чтобы ты понял контекст проекта?
```

## 2) “Сделай README проекта”

```
Сделай README.md для проекта.
В README должны быть:
- что делает проект
- как установить зависимости
- как запустить
- как прогнать тесты
- env vars (без секретов)

Вот структура проекта:
[вставь дерево папок или список файлов]
```

## 3) “Проверь, достаточно ли контекста”

```
Вот README и структура проекта:
[вставь]

Скажи, чего не хватает, чтобы новый разработчик смог запустить проект с нуля.
Дай список конкретных правок.
```
