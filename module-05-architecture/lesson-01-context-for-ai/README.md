## 📖 Теория (минимум)

AI даёт лучшие ответы, когда получает:
- **структуру проекта** (папки, точка входа, где конфиги),
- **как запускать** (команды),
- **ограничения** (версии, что нельзя менять),
- **цель** (что считаем “готово”).

### “Пакет контекста” (что положить в README проекта)

- Что это за проект (1–2 абзаца)
- Как запустить локально
- Переменные окружения (без секретов)
- Структура папок и где искать ключевой код
- Типовые команды (тесты, линтер, Docker)
