# Шпаргалка: контекст для AI (урок 5.1)

## Минимальный “контекст-пакет” (в README проекта)

- Что делает проект (1–2 абзаца)
- Как запустить локально (команды)
- Где точка входа (main/app) и основные папки
- Как прогнать тесты/линтер
- Какие есть env vars (без секретов)
- Что нельзя менять (ограничения)

## 5 файлов, которые чаще всего нужны AI

- `README.md`
- `requirements.txt` / `package.json`
- точка входа (`main.py`/`app.py`/`src/...`)
- конфигурация (`.env.example`, `config.*`)
- CI (`.github/workflows/*`)
