# Домашнее задание: Урок 5.1 (Контекст для AI)

## Задание: улучшить контекст проекта и проверить эффект

1) Выберите проект (учебный или свой).  
2) Добавьте/обновите README так, чтобы в нём были:
- что делает проект,
- как запустить,
- структура папок,
- как прогнать тесты,
- список env vars (без секретов).
3) Попросите AI сделать одну правку в проекте **до** и **после** добавления контекста.
4) Сравните качество: что изменилось (3–7 пунктов).

## Критерии сдачи

- [ ] README стал пригоден для “git clone → run”
- [ ] Есть сравнение “до/после” с выводами
