# Переменные в Python

*По материалам [W3Schools: Python Variables](https://www.w3schools.com/python/python_variables.asp)*

## Что такое переменная?

Переменная — это контейнер для хранения данных. В Python переменная создаётся в момент присваивания значения:

```python
x = 5
name = "Анна"
```

## Создание переменных

Python не требует объявления типа. Тип определяется значением:

```python
x = 4       # int
y = 5.5     # float
name = "Иван"  # str
```

## Имена переменных

- Должны начинаться с буквы или подчёркивания
- Не могут начинаться с цифры
- Содержат только буквы, цифры и подчёркивания
- Регистр важен: `age` и `Age` — разные переменные

**Хорошие имена:**
```python
my_var = 1
user_name = "Анна"
_count = 0
```

**Плохие имена:**
```python
2name = "ошибка"   # нельзя начинать с цифры
my-var = 1         # дефис недопустим
```

## Присваивание нескольких значений

```python
x, y, z = 1, 2, 3
a = b = c = 0
```

## Вывод переменных

```python
name = "Мир"
print("Привет,", name)
print(f"Привет, {name}!")
```

## Глобальные и локальные переменные

Переменная внутри функции — локальная. Снаружи — глобальная. Ключевое слово `global` позволяет изменять глобальную переменную из функции:

```python
x = 10

def my_func():
    global x
    x = 20

my_func()
print(x)  # 20
```

## Важно для работы с AI

AI часто забывает, что `input()` возвращает строку. Всегда проверяйте типы и преобразуйте при необходимости:

```python
age = input("Возраст: ")   # age — строка!
age = int(age)             # преобразуем в число
```
