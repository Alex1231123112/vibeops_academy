# Матрица навыков (Junior-ready)

Цель: выпускник способен **самостоятельно** разработать небольшой сервис, протестировать, упаковать в Docker, настроить CI/CD и поддерживать минимальный прод (логи/health/secrets).

Формат: **урок → навыки → проверка → артефакт**.

---
