# Анализ полноты наполнения курса

Документ фиксирует текущее состояние материалов и возможные дополнения. Обновляется по мере изменений.

## Критерии «полноты» урока

- **README** с целями, теорией и примерами (или ссылками на них).
- **Домашнее задание** (homework.md) или явное упражнение в README.
- **Промпты для AI** (prompts.md) — опционально, но желательно для уроков с кодом.
- **Примеры кода** (examples/) — где уместно (тестирование, файлы, и т.д.).
