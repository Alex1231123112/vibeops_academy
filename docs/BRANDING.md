# Брендинг VibeOps Академия

## Описание проекта

### Короткое (для описания канала Telegram, до ~255 символов)

```
Бесплатный курс: программирование с AI + DevOps. От первой переменной до приложения в облаке. Python, Git, Docker, CI/CD. Промпты, Cursor, понимание кода — без зависимости от нейросетей.
```

### Среднее (для поста, лендинга)

**VibeOps Академия** — бесплатный курс для тех, кто хочет программировать с помощью AI и освоить DevOps. Вы пройдёте путь от основ Python до деплоя в облако: Git, Docker, CI/CD, промпт-инжиниринг. Учим понимать код, а не только генерировать. 6 модулей, 20+ материалов, финальный проект. Для новичков и разработчиков без DevOps-опыта.

### Ключевые слова

Vibe Coding, AI-разработка, Python, DevOps, Docker, CI/CD, Cursor, промпты, обучение программированию, бесплатный курс.

## Промпт для генерации аватарки канала (Telegram)

Используйте в Midjourney, DALL·E, Stable Diffusion или другом генераторе. Аватар в Telegram обрезается по кругу; оптимально — квадрат 512×512 или 1024×1024.

### Вариант 1 (минималистичный, иконка)

```
Modern flat icon for an educational tech brand. Minimalist logo: abstract combination of code bracket and gear or terminal window, symbolizing coding and DevOps. Colors: deep dark background (#0f0f0f), accent cyan/teal (#26A5E4). Clean, professional, no text. Square format, centered, suitable for app icon and Telegram channel avatar.
```

```
Digital illustration for a programming education channel. Style: sleek, modern, dark theme. A subtle glow of code lines or circuit pattern in the background, soft cyan and blue accents (#26A5E4). Abstract, not literal. Mood: learning, technology, clarity. Square composition, no text, works as round avatar.
```

### Вариант 3 (краткий универсальный)

```
Tech education logo, minimalist. Code + DevOps symbol, dark background, cyan accent. No text. Square, clean, professional avatar for Telegram channel.
```

## Где подставить описание

- **Telegram-канал:** Настройки канала → Описание — вставьте короткое описание.
- **Чат:** Описание группы/чата — можно то же короткое или «Обсуждение курса VibeOps Академия».
- **Аватарка:** Сгенерируйте по промпту выше, сохраните квадрат 512×512 или больше, загрузите в настройках канала как фото профиля.
