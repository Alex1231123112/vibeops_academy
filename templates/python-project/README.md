# Python Project Template

Базовый шаблон для Python-проекта.

## Структура

- main.py — точка входа
- requirements.txt — зависимости
- Dockerfile — контейнеризация
- .gitignore — игнорируемые файлы

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Docker

```bash
docker build -t my-app .
docker run my-app
```
