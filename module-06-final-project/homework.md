# Домашнее задание: Модуль 6 (Финальный проект)

## Задание: проект от идеи до продакшна

1) Выберите идею из списка или придумайте свою: [project-ideas.md](project-ideas.md)  
2) Возьмите шаблон и соберите структуру проекта: [project-template/](project-template/)  
3) Реализуйте минимальный функционал (MVP)  
4) Доведите проект до чек-листа “junior-ready”  

## Что обязательно сделать

- README: что это, как запустить, как прогнать тесты
- `/health` endpoint
- Базовые логи
- Dockerfile + `.dockerignore`
- docker-compose (поднимает проект)
- CI: тесты на `push/pull_request`
- CD: деплой на `main` (или чёткая инструкция деплоя, если без автодеплоя)
- HTTPS URL

## Формат сдачи

- ссылка на GitHub репозиторий
- ссылка на задеплоенный сервис (HTTPS)
- короткий отчёт (5–10 строк): что сделали, как проверяли
