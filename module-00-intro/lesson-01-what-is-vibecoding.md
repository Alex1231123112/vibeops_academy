## 📖 Теория

### Что такое Vibe Coding?

**Vibe Coding** — подход к программированию, когда вы используете AI (ChatGPT, Cursor, Claude Code, Copilot) как инструмент для написания кода, сохраняя контроль и понимание. Термин стал популярным в 2024–2025.

### Два полюса

| AI-зависимый | AI-инженерный |
|--------------|---------------|
| Копируешь код без понимания | Понимаешь, что делает каждая строка |
| "Не работает — не знаю почему" | Умеешь отлаживать и исправлять |
| Нет версионирования | Git, история изменений |
| Запуск на своём компьютере | Деплой в облако |
| Один раз сработало | Воспроизводимая среда |

### Ключевая идея

> AI не заменит инженера. AI заменит инженера, который не умеет AI.

Vibe Coding — это не "просто копировать код из ChatGPT". Это умение:
- Формулировать задачи для AI
- Проверять и корректировать результат
- Интегрировать код в проект
- Разворачивать и поддерживать приложение

## 🤖 Зачем это нужно?

1. **AI ошибается** — нужно уметь находить и исправлять ошибки
2. **AI не знает контекст** — нужно понимать архитектуру
3. **Код должен работать** — нужны CI/CD, Docker, тесты
4. **Работодатели ценят** — умение работать с AI + DevOps = востребованный специалист
