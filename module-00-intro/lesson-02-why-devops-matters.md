## 📖 Теория

### DevOps — это не только про серверы

В контексте Vibe Coding DevOps — это набор навыков, которые позволяют:

1. **Версионировать код** — Git, чтобы не потерять работу и откатиться
2. **Создавать окружение** — Docker, чтобы "у меня работает" стало "у всех работает"
3. **Автоматизировать деплой** — CI/CD, чтобы каждый push = обновление в облаке
4. **Понять инфраструктуру** — порты, домены, HTTPS

### Почему это важно для AI-разработчиков?

| Проблема | DevOps-решение |
|----------|----------------|
| "У меня работает, у тебя нет" | Docker — одинаковое окружение |
| "Потерял код" | Git — вся история в репозитории |
| "Как выложить в интернет?" | Render, Railway — деплой в 5 минут |
| "AI сгенерировал 1000 строк — как разобраться?" | Структура проекта, модули, тесты |

### Минимальный набор для старта

1. **Git** — 10 команд, которые нужно знать
2. **Терминал** — навигация, запуск скриптов
3. **Docker** — упаковать приложение в контейнер
4. **GitHub Actions** — автоматический деплой
