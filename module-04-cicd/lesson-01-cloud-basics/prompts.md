# Промпты для AI: Урок 4.1 (Облачные основы)

## 1) “Как задеплоить мой проект?”

```
Я хочу задеплоить проект на [Render/Railway].
Вот стек: [FastAPI/Flask/Node/другое].
Вот как запускается локально: [команда].

Составь пошаговую инструкцию:
- что выбрать в интерфейсе
- какие команды build/start указать
- какие env vars нужны
- как проверить деплой (URL, health, логи)
```
