# Шпаргалка: облачный деплой (урок 4.1)

## Минимальные элементы деплоя (PaaS)

- Репозиторий на GitHub
- Команда сборки (если нужна)
- Команда запуска
- Порт (если платформа требует)
- Переменные окружения (secrets)
- Логи (build + runtime)

## Быстрая проверка “всё живо”

- Деплой завершился успешно (в UI зелёный статус)
- URL открывается в браузере
- `/health` возвращает `200` и JSON (если есть)
- В логах нет циклических падений/рестартов
