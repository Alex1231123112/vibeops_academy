# Домашнее задание: Урок 4.1 (Облачные основы)

## Задание: первый деплой на Render или Railway

1) Выберите платформу: **Render** или **Railway**.  
2) Подключите GitHub-репозиторий с вашим проектом.  
3) Настройте деплой:
- команда запуска,
- переменные окружения (если есть),
- порт (если нужно).
4) Дождитесь успешного деплоя и получите публичный URL.  
5) Проверьте, что приложение отвечает (хотя бы главная страница или `/health`).

## Критерии сдачи

- [ ] Есть ссылка на задеплоенное приложение
- [ ] Вы можете показать, где смотреть логи деплоя и приложения
- [ ] (Желательно) Есть `/health` и вы проверили его вручную
