## 📖 Теория (минимум)

### Что такое PaaS (простыми словами)

Render/Railway — это платформы, где вы деплоите код, а они:
- собирают и запускают приложение,
- дают URL,
- управляют переменными окружения,
- показывают логи.

### Что важно при выборе

- Бесплатный тариф (лимиты, “засыпание”)
- Наличие автодеплоя из GitHub
- Настройка переменных окружения (secrets)
- Поддержка Docker (если проект контейнерный)
