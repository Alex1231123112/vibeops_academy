# Шпаргалка: HTTPS и домены (урок 4.3)

## Минимум терминов

- **DNS** — “телефонная книга” интернета: домен → IP/имя сервиса
- **A запись** — домен → IPv4 адрес
- **CNAME** — домен → другое доменное имя
- **TLS сертификат** — нужен для HTTPS

## Типовой сценарий на PaaS

1) Добавить домен в UI платформы  
2) Добавить DNS записи у регистратора (обычно CNAME)  
3) Подождать обновления DNS (минуты/часы)  
4) Платформа выпускает сертификат автоматически  

## Быстрая диагностика

- Проверьте, что DNS записи введены без лишних точек/префиксов
- Убедитесь, что нет конфликтующих A/CNAME записей на один и тот же хост
- Дайте время на TTL / распространение DNS
