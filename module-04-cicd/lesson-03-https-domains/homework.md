# Домашнее задание: Урок 4.3 (HTTPS и домены)

## Задание A (обязательно): HTTPS URL

1) Возьмите задеплоенный сервис (из 4.1 или 4.4).  
2) Убедитесь, что он доступен по **HTTPS**.  
3) Проверьте `/health` (или другую страницу/эндпоинт).

## Задание B (желательно): свой домен

1) Подключите домен к вашему сервису на платформе (Render/Railway/другая).  
2) Добавьте нужные DNS записи у регистратора.  
3) Дождитесь, пока сертификат выпустится, и проверьте HTTPS.

## Критерии сдачи

- [ ] Есть HTTPS ссылка на сервис
- [ ] (Желательно) Есть подключенный домен
- [ ] Вы можете объяснить, какие DNS записи добавляли (CNAME/A)
