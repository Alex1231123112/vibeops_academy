## 📖 Теория (минимум)

### Что такое домен

Домен — это имя, которое указывает на ваш сервис (через DNS).

### Что такое HTTPS

HTTPS = HTTP + шифрование + сертификат (TLS). Для учебного проекта важно:
- чтобы сервис открывался по HTTPS,
- чтобы браузер “не ругался”.

### Где обычно включается HTTPS

На PaaS (Render/Railway) чаще всего HTTPS включён автоматически на их домене.
