# Шпаргалка: Терминал (Linux/macOS)

| Команда | Описание |
|---------|----------|
| `pwd` | Текущая директория |
| `ls` | Список файлов |
| `cd <path>` | Перейти в папку |
| `mkdir <name>` | Создать папку |
| `rm <file>` | Удалить файл |
| `rm -r <dir>` | Удалить папку рекурсивно |
| `python script.py` | Запустить Python |
| `python3 script.py` | Запустить Python 3 |
