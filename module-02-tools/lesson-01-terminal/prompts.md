# Промпты для AI: Урок 2.1 (Терминал)

## 1) “Какая команда в терминале?”

```
Я на Windows PowerShell (или Linux bash).
Мне нужно сделать: [опиши действие].
Дай команды и объясни, что они делают. Если команды отличаются для Windows и Linux — покажи оба варианта.
```

## 2) “Почему команда не работает?”

```
Команда не работает.
Команда: [вставь команду]
Вывод ошибки: [вставь вывод]

1) Объясни причину простыми словами
2) Дай 2–3 варианта исправления
3) Скажи, как проверить, что я нахожусь в правильной папке
```

## 3) “Как запустить Python-скрипт?”

```
У меня есть файл main.py. Я хочу запустить его из терминала.
Подскажи:
- как проверить версию Python
- как запустить файл
- что делать, если команда python не находится
Я на: [Windows PowerShell / macOS / Linux]
```
