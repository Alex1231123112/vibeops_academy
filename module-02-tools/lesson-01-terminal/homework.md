# Домашнее задание: Терминал

## Задание: “папка проекта” и запуск скрипта

1) Создайте папку `my-project`  
2) Перейдите в неё  
3) Создайте файл `main.py` с кодом:

```python
print("Hello from terminal!")
```

4) Запустите скрипт из терминала:
- `python main.py`

## Дополнительно (желательно)

- Создайте ещё одну папку `data/` и файл `data/notes.txt`
- Проверьте, что вы можете:
  - перейти в `data/` и вернуться обратно (`cd ..`)
  - вывести содержимое текущей папки (`ls`)

## Критерии сдачи

- [ ] В репозитории/папке есть `my-project/main.py`
- [ ] Скрипт запускается командой `python main.py` и печатает строку
- [ ] Вы понимаете, в какой папке находитесь (можете показать `pwd`)
