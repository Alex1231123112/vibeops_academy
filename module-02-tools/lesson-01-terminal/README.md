## 📖 Теория (минимум)

### Ментальная модель

- **Терминал** — способ управлять компьютером командами.
- Вы всегда “стоите” в какой-то папке (текущая директория) и выполняете команды относительно неё.

### Базовые команды (Linux/macOS и аналоги в PowerShell)

| Задача | Linux/macOS | PowerShell (Windows) |
|---|---|---|
| Где я? | `pwd` | `pwd` |
| Что в папке? | `ls` | `ls` |
| Перейти в папку | `cd path` | `cd path` |
| Создать папку | `mkdir name` | `mkdir name` |
| Удалить файл | `rm file` | `rm file` |
| Удалить папку | `rm -r dir` | `rm -r dir` |
| Создать файл быстро | `touch a.txt` | `New-Item a.txt` |

### Запуск Python

- `python --version` — проверить версию
- `python main.py` — запустить файл
