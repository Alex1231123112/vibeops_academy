# Git Flow для начинающих

1. `git init` — один раз при создании проекта
2. `git add .` — добавить изменения
3. `git commit -m "описание"` — зафиксировать
4. Создать репозиторий на GitHub
5. `git remote add origin <url>`
6. `git push -u origin main` — отправить

Далее при изменениях: `add` → `commit` → `push`
