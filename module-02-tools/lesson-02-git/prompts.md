# Промпты для AI: Урок 2.2 (Git)

## 1) “Что мне сделать в Git пошагово?”

```
Я хочу: [опиши цель — например: создать репо, запушить, сделать PR].
Я на: [Windows/macOS/Linux].

Дай пошаговые команды Git и объясни, что делает каждый шаг.
Если есть типичные ошибки — предупреди (например, забыли git add, нет origin, нет прав).
```

## 2) “Разобрать git status и исправить”

```
Вот мой вывод:
git status:
[вставь]

Что это значит и какие команды мне выполнить дальше?
```

## 3) “Конфликт при merge”

```
У меня конфликт при merge/PR.
Вот сообщение:
[вставь]

Объясни, что это значит, как его решить и как проверить, что всё ок перед коммитом.
```
