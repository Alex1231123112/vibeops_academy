# Домашнее задание: Git

## Задание: репозиторий + история изменений

1) Создайте репозиторий на GitHub  
2) Клонируйте его локально  
3) Добавьте файл `README.md` (1–2 абзаца: о чём проект)  
4) Сделайте `commit` и `push`

## Дополнительно (желательно): ветка и PR

1) Создайте ветку `feature/readme-update`  
2) Измените `README.md` (добавьте раздел “Как запустить”)  
3) Сделайте `commit` в ветке и `push`  
4) Откройте Pull Request в GitHub и слейте его (Merge)

## Критерии сдачи

- [ ] Есть ссылка на репозиторий на GitHub
- [ ] В истории минимум 2 коммита
- [ ] (Желательно) Был PR из feature-ветки в `main`
