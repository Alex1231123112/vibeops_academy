# Домашнее задание: Урок 2.5 (Зависимости)

## Задание: venv + requirements.txt

1) Создайте папку проекта (или возьмите любой учебный проект).  
2) Создайте виртуальное окружение:
- `python -m venv .venv`
3) Активируйте его (Windows / Linux/macOS).  
4) Установите 1–2 пакета (например `requests`, `pytest`).  
5) Зафиксируйте зависимости в `requirements.txt`.

## Проверка “с нуля” (важно)

1) Удалите окружение `.venv` (или создайте новую папку-копию проекта).  
2) Создайте и активируйте новое окружение.  
3) Установите зависимости из `requirements.txt`.  
4) Убедитесь, что проект запускается.

## Критерии сдачи

- [ ] В проекте есть `.venv/` (или вы понимаете, как её создать)
- [ ] Есть `requirements.txt`
- [ ] Вы проверили воспроизводимость (создали окружение заново и всё запустилось)
