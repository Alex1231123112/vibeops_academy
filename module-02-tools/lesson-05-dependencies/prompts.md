# Промпты для AI: Урок 2.5 (Зависимости)

## 1) “Какие зависимости нужны?”

```
Я хочу сделать: [описание — например, HTTP запросы / FastAPI / тесты].
Подскажи:
- какие пакеты нужны
- какие команды выполнить (venv + pip install)
- как оформить requirements.txt
Моя ОС: [Windows/macOS/Linux]
```

## 2) “Не ставится пакет / конфликт версий”

```
pip install не работает.
Команда:
[вставь]
Ошибка:
[вставь]

Объясни причину и предложи шаги исправления.
Отдельно: как проверить, что я в активированном venv?
```

## 3) “Сделай инструкцию запуска проекта”

```
У меня проект на Python.
Составь короткую инструкцию запуска:
- создать venv
- установить зависимости
- запустить приложение/тесты
Дай команды для Windows PowerShell и для Linux/macOS.
```
