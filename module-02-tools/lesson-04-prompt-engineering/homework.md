# Домашнее задание: Урок 2.4 (Промпт-инжиниринг)

## Задание: улучшить 2 промпта и сравнить результат

Выберите **две** задачи из списка:
- исправить ошибку в коде,
- рефакторинг (вынести в функции, улучшить читаемость),
- написать тесты,
- написать Dockerfile/CI workflow.

Для каждой задачи сделайте:

1) **Плохой промпт** (короткий, без контекста)  
2) **Хороший промпт** по структуре: контекст → цель → ограничения → формат ответа → критерии  
3) Сравнение результатов: 3–7 пунктов “что улучшилось”

## Формат сдачи

Один `md` файл или сообщение в чате:
- Задача 1: промпт A → промпт B → сравнение
- Задача 2: промпт A → промпт B → сравнение

## Критерии сдачи

- [ ] Есть 2 задачи
- [ ] Для каждой задачи есть “плохой” и “хороший” промпт
- [ ] Есть сравнение результатов
