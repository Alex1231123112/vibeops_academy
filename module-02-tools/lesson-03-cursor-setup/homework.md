# Домашнее задание: Урок 2.3 (Cursor)

## Задание A (обязательно): открыть проект и использовать AI-правки

1) Установите Cursor и откройте папку любого проекта (можно ваш учебный проект из модуля 1).  
2) Проверьте, что работает:
- чат (обсудить задачу),
- inline-редактирование (`Ctrl+K`),
- встроенный терминал.

3) Попросите AI сделать **одну небольшую правку** в коде (например: переименовать переменные, вынести кусок в функцию, улучшить обработку ошибок).

## Задание B (желательно): контекст проекта для AI

1) Создайте в проекте файл с правилами/контекстом (например, “как называется точка входа”, “какая структура папок”, “что нельзя делать”).  
2) Проверьте, что AI учитывает этот контекст: попросите сделать ещё одну правку и сравните качество ответа.

## Критерии сдачи

- [ ] Есть проект, который открывается в Cursor
- [ ] Вы использовали чат и inline-редактирование хотя бы 1 раз
- [ ] Есть 1–2 изменения в коде, которые вы понимаете и можете объяснить
