# Шпаргалка: Cursor (урок 2.3)

## Горячие клавиши (база)

| Действие | Клавиши |
|---|---|
| Inline-редактирование с AI | `Ctrl+K` |
| Открыть чат | `Ctrl+L` |
| Встроенный терминал | `Ctrl+`` |

## Быстрые паттерны работы

- **Правка кода**: выдели фрагмент → `Ctrl+K` → “сделай X, не меняй поведение”.
- **Понять контекст**: в чат → “объясни этот модуль и точки входа”.
- **Декомпозиция**: “сначала задай 5 уточняющих вопросов, потом план”.
