# Промпты для AI: Урок 2.3 (Cursor)

## 1) “Сделай правку, но объясни”

```
Вот код/файл:
[вставь]

Сделай правку: [опиши что нужно].
Требования:
- сохрани поведение
- объясни изменения коротко (3–7 пунктов)
- покажи, что именно менять (патч/фрагменты)
```

## 2) “Какие файлы мне показать для контекста?”

```
Я хочу, чтобы ты помог с задачей: [задача].
Какие 3–5 файлов и папок мне нужно показать, чтобы ты понял контекст проекта?
```

## 3) “Проверка результата”

```
Ты предложил изменения.
Составь чек-лист: как проверить, что ничего не сломалось (команды, ручные проверки, edge cases).
```
