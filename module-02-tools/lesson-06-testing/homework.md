# Домашнее задание: Урок 2.6 (pytest)

## Задание A (обязательно): Unit-тесты

1) Создайте файл `calc.py` с функциями:
- `add(a: int, b: int) -> int`
- `divide(a: float, b: float) -> float` (с обработкой деления на 0 через `ValueError`)

2) Создайте `test_calc.py` и напишите тесты:
- `add(2, 3) == 5`
- `divide(10, 2) == 5`
- `divide(10, 0)` выбрасывает `ValueError`

## Задание B (желательно): API smoke-тест

Если у вас уже есть маленький FastAPI/Flask проект — добавьте один тест, который дергает `/health` и проверяет `200` и JSON.

## Критерии сдачи

- [ ] Тесты запускаются локально: `pytest`
- [ ] В репозитории есть папка `tests/` или файл `test_*.py`
- [ ] Есть GitHub Actions workflow, который запускает тесты на `push` и `pull_request`
