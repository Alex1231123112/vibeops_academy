# Роли в проекте VibeOps

## 👥 Роли людей

| Роль | Ответственность | Как получить |
|------|-----------------|--------------|
| **Maintainer** | Релизы, roadmap, merge PR, координация | По приглашению |
| **Contributor** | Уроки, примеры кода, исправления | Открыть PR |
| **Reviewer** | Ревью PR, проверка качества | По запросу / добровольно |
| **Community** | Вопросы в чате, помощь новичкам | Участие в @vibeops_academy_chat |

## 🤖 Роли AI-агентов (agency-agents)

При разработке курса каждый тип задачи закреплён за агентом.

### Контент

| Роль | Агент | Задачи |
|------|-------|--------|
| **Автор уроков** | Content Creator | Тексты, структура, объяснения |
| **Редактор** | UX Researcher | Читаемость, понятность для новичков |

### Код

| Роль | Агент | Задачи |
|------|-------|--------|
| **Архитектор** | Backend Architect | API, структура, примеры |
| **Прототипер** | Rapid Prototyper | Быстрые примеры, MVP-код |
| **Фронтенд** | Frontend Developer | Шаблоны web-app, UI-примеры |

### Инфраструктура

| Роль | Агент | Задачи |
|------|-------|--------|
| **DevOps** | DevOps Automator | Dockerfile, CI/CD, пайплайны |
| **Безопасность** | Security Engineer | Ревью кода, best practices |

### Контроль качества

| Роль | Агент | Задачи |
|------|-------|--------|
| **Проверка** | Reality Checker | Готовность к merge, продакшн |
| **Тестирование** | Evidence Collector | Верификация примеров |

### Координация

| Роль | Агент | Задачи |
|------|-------|--------|
| **Оркестратор** | Agents Orchestrator | Сложные задачи с несколькими агентами |

## 📋 Матрица: задача → агент

| Задача | Первичный агент | Дополнительно |
|--------|-----------------|---------------|
| Новый урок | Content Creator | — |
| Пример Python | Backend Architect | Rapid Prototyper |
| Dockerfile | DevOps Automator | Security Engineer |
| GitHub Actions | DevOps Automator | — |
| API-дизайн | Backend Architect | — |
| Ревью PR | Reality Checker | Security Engineer |
| Шаблон проекта | Rapid Prototyper | Backend Architect |
