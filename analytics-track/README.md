# 📊 Analytics Track: Аналитик с AI (от нуля до junior-ready)

Это отдельный трек курса VibeOps для **всех основных ролей аналитиков**. Трек построен так, чтобы:
- дать общее ядро (инструменты, SQL, Python для данных, визуализация, метрики, качество данных);
- затем развести обучение по специализациям;
- привести к финальному проекту с понятными артефактами (репо + воспроизводимый запуск + “публичный результат”).

## 🎯 Для кого

- Product / Growth Analyst
- BI Analyst
- Data Analyst (универсальный)
- Marketing Analyst
- Financial Analyst
- Risk / Fraud Analyst
- Data Scientist (entry / DS-lite)
- Analytics Engineer (dbt-style)

## Как устроен трек

### 1) Core (общее ядро)

Папка: [core/](core/)

Содержание: базовые инструменты и навыки, которые нужны всем аналитикам.

### 2) Специализации (ветки)

Папка: [specializations/](specializations/)

Каждая ветка добавляет 3–5 уроков и отдельный финальный проект.

## ⏱️ Сколько времени

- Core: **12–20 часов**
- Специализация: **6–12 часов**
- Финальный проект: **6–12 часов**

Итого ориентир: **24–44 часа** (зависит от роли и глубины).

## ✅ Артефакты (что должно получиться)

- GitHub репозиторий с анализом (структура проекта, `requirements.txt`, инструкции запуска)
- SQL/ноутбуки/скрипты с воспроизводимым запуском
- Базовые data checks (валидность типов/диапазонов/пропусков)
- (по специализации) дашборд/отчёт/модель/пакет метрик
- CI, который делает минимум: линт/проверки/тесты (по возможности)
