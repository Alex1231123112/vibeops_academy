# Финальный проект: Risk/Fraud Analyst

## Задача

Сделать baseline-детектор аномалий (правила/простая модель) + мониторинг метрик.

## ✅ Критерии

- [ ] есть детектор (правила/скоринг)
- [ ] есть оценка качества (precision/recall или proxy) на тестовых данных
- [ ] есть мониторинг метрики “аномалий” во времени
- [ ] есть рекомендации по улучшению (3–5 пунктов)
