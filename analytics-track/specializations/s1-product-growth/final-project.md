# Финальный проект: Product/Growth

## Задача

Собрать отчёт по продуктовым событиям: воронка + retention + 3 рекомендации.

## Артефакты

- `reports/report.md` (или дашборд + ссылка)
- запросы/код, который воспроизводит расчёты
- definitions: словарь метрик/событий (1 файл)

## ✅ Критерии

- [ ] есть воронка минимум из 3 шагов
- [ ] есть когорты retention (минимум 3 когорты)
- [ ] есть 3 рекомендации “что делать дальше”
- [ ] анализ воспроизводим по README
