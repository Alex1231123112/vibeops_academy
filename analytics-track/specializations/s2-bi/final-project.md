# Финальный проект: BI Analyst

## Задача

Собрать дашборд KPI + словарь метрик.

## Артефакты

- KPI dictionary (Markdown)
- дашборд (ссылка/скрин) или отчёт с графиками
- описание источников данных и обновления

## ✅ Критерии

- [ ] у KPI есть определения (формула, период, сегмент)
- [ ] есть минимум 6 KPI на дашборде/в отчёте
- [ ] есть “как обновлять данные”
