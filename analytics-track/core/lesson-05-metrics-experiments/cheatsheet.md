# Шпаргалка: метрики и эксперименты (A5)

## Определение метрики

- **Что считаем** (событие/состояние)
- **Кого считаем** (сегмент)
- **Период**
- **Формула** (числитель/знаменатель)

## Воронка

- шаги: `view → signup → purchase`
- считать конверсию между шагами

## Когорты

- определить когорту (например, неделя регистрации)
- сравнивать retention по когортам

## A/B риски (минимум)

- peeking (подсматриваем слишком рано)
- множественные проверки
- разная структура трафика/сезонность
