# A5: Метрики и эксперименты (основа)

## 🎯 Цели урока

- Понимать продуктовые метрики (конверсия, retention, LTV/CAC — на уровне базы)
- Уметь строить воронку и когорту
- Понимать основы A/B: гипотеза, метрика успеха, риски интерпретации

## ⏱️ Сколько времени

Ориентир: **3–6 часов**.

## 📺 Видео

- [Все видео курса](../../../resources/videos.md)

## 📖 Теория (минимум)

- Метрика должна иметь **определение** (числитель/знаменатель/период/сегмент)
- A/B — это не “посмотрели на рост”, а сравнение групп + контроль рисков (peeking, множественные проверки)
