# Промпты для AI: A5 (метрики/эксперименты)

База: [prompts-library/template.md](../../../prompts-library/template.md)

## 1) “Определи метрику формально”

```
Мне нужна метрика: [описание].
Сформулируй определение:
- числитель/знаменатель
- сегмент
- период
- какие события/таблицы нужны
```

## 2) “Список рисков интерпретации”

```
Вот результат анализа:
[вставь]

Перечисли 7 рисков интерпретации и как проверить каждый.
```
