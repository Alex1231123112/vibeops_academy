# Домашнее задание: A5 (метрики)

## Задание: метрики + воронка/когорта

1) Опишите 3 метрики формально (определение + формула + сегмент + период).  
2) Постройте:
- простую воронку (3 шага), или
- когорту retention (2–3 когорты).

## Критерии сдачи

- [ ] Есть 3 метрики с формальными определениями
- [ ] Есть воронка или когорта + короткий вывод
- [ ] Перечислены 2–3 риска интерпретации результата
