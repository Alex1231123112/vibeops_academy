# Промпты для AI: A1 (инструменты аналитика)

База: [prompts-library/template.md](../../../prompts-library/template.md)

## 1) Сгенерируй README для анализа

```
Сделай README.md для аналитического проекта.
В README должны быть:
- цель анализа
- источник данных
- как установить зависимости (venv)
- как запустить ноутбук/скрипт
- как воспроизвести результаты
```

## 2) “Какие файлы показать для контекста?”

```
Я делаю анализ.
Задача: [описание]

Какие 5–10 файлов/папок мне показать, чтобы ты помог качественно?
```
