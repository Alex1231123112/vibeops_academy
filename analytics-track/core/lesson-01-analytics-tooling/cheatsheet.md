# Шпаргалка: аналитический проект (A1)

## Минимальная структура

```
analysis-project/
  README.md
  requirements.txt
  data/           (сырые данные или ссылки)
  notebooks/      (*.ipynb)
  src/            (переиспользуемый код)
  reports/        (markdown/pdf, графики)
```

## Минимальные команды (Python)

- `python -m venv .venv`
- activate
- `pip install -r requirements.txt`
- `pip freeze > requirements.txt`
