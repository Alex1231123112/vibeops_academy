# Шпаргалка: data quality (A6)

## Минимальные проверки

- schema: “колонки существуют и типы ожидаемые”
- null: “в ключевых полях нет NULL”
- range: “значения в разумных пределах”
- uniqueness: “id уникален”
- referential: “ссылки на сущности существуют”

## Где фиксировать проверки

- в коде (python)
- в SQL (отдельные запросы)
- в CI (как отдельный шаг)
