# Домашнее задание: A6 (качество данных)

## Задание: набор data checks

Сделайте 5–10 проверок качества данных для вашего датасета:

- 2 schema checks (колонки/типы)
- 2 null checks
- 2 range checks
- 1 uniqueness check
- (опционально) 1 check “отношений” (например, все user_id существуют в справочнике)

Оформите как:
- python скрипт, или
- набор SQL запросов, или
- оба варианта.

## Критерии сдачи

- [ ] 5+ checks
- [ ] Понятно, что проверяет каждый check (1 строка описания)
- [ ] Есть команда запуска checks (в README)
