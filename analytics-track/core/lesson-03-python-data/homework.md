# Домашнее задание: A3 (pandas)

## Задание: мини-пайплайн обработки данных

Сделайте скрипт или ноутбук, который:
1) читает CSV,
2) чистит данные (типы/пропуски),
3) делает merge (если есть 2 таблицы) или группировку,
4) считает 3 метрики,
5) сохраняет результат в `reports/` (CSV или Markdown).

## Критерии сдачи

- [ ] Есть воспроизводимый запуск (команды в README)
- [ ] Есть 3 метрики и короткие выводы
- [ ] Есть минимум 2 проверки качества (NULL/диапазоны/дубликаты)
