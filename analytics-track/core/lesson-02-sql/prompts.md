# Промпты для AI: A2 (SQL)

База: [prompts-library/template.md](../../../prompts-library/template.md)

## 1) Сгенерируй запрос + проверки качества

```
У меня есть задача:
[описание]

Схема таблиц:
[вставь таблицы и поля]

Сгенерируй SQL запрос.
Требования:
- поясни кардинальность JOIN
- добавь 2 проверки качества (до/после JOIN, NULL, дубликаты)
```

## 2) Отладка запроса

```
Вот запрос:
[вставь]

Он даёт странный результат: [описание]
Найди ошибку и предложи исправление.
```
