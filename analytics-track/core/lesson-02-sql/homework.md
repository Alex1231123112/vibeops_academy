# Домашнее задание: A2 (SQL)

## Задание: 6 запросов “как в работе”

На любом датасете (можно учебном) сделайте 6 запросов:

1) SELECT + фильтр по периоду  
2) JOIN 2 таблиц (и объяснить кардинальность)  
3) GROUP BY + top-N  
4) CTE (`WITH`)  
5) Оконная функция (например `ROW_NUMBER()` или `COUNT() OVER`)  
6) Качество данных: запрос, который ищет проблему (дубликаты/NULL/аномалии)

## Критерии сдачи

- [ ] Есть 6 запросов
- [ ] Вы написали 2–5 строк “что проверяет каждый запрос”
- [ ] Есть минимум 2 проверки качества (до/после JOIN, NULL, дубликаты)
