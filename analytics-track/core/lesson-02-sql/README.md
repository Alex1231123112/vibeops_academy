## 🎯 Цели урока

- Писать SELECT/JOIN/GROUP BY для реальных задач
- Использовать CTE и оконные функции (основа)
- Понимать качество данных и типовые ошибки (дубликаты, пропуски, неверные джоины)

## 📖 Теория (минимум)

Минимум, который нужен в работе:
- `JOIN` и “кардинальность” (1-1, 1-N, N-N)
- `GROUP BY` и агрегаты
- CTE (`WITH ...`)
- оконные функции (`ROW_NUMBER`, `COUNT OVER`, `SUM OVER`) — на уровне “уметь читать/применять”
