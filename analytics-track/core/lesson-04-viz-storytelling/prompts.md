# Промпты для AI: A4 (визуализация)

База: [prompts-library/template.md](../../../prompts-library/template.md)

## 1) “Какой график лучше?”

```
У меня задача:
[описание]

Данные:
[колонки и примеры]

Какой тип графика лучше и почему? Дай код (matplotlib или plotly) и список ошибок визуализации, которых избегать.
```

## 2) “Сформулируй выводы”

```
Вот график/таблица (описание):
[вставь]

Сформулируй 3 вывода:
- что произошло
- почему важно
- что делать
Коротко и по делу.
```
