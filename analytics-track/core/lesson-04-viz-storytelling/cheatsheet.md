# Шпаргалка: графики (A4)

## Какой график выбрать

- **Динамика** → line
- **Сравнение категорий** → bar
- **Распределение** → histogram / box
- **Связь двух метрик** → scatter

## Мини-чек на качество

- Есть заголовок + оси + единицы измерения
- Нет обрезки оси Y “чтобы усилить эффект”
- Подписи читаемые, легенда понятная
