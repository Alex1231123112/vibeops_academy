# A4: Визуализация и сторителлинг

## 🎯 Цели урока

- Выбирать правильный тип графика под задачу
- Делать понятные подписи/оси/легенды и “не врать графиками”
- Писать выводы: что произошло, почему это важно, что делать

## ⏱️ Сколько времени

Ориентир: **2–4 часа**.

## 📺 Видео

- [Все видео курса](../../../resources/videos.md)

## 📖 Теория (минимум)

- line: динамика во времени
- bar: сравнение категорий
- histogram/box: распределения
- scatter: связь двух величин

Главное правило: **один график — одна мысль**.
