# Домашнее задание: A4 (визуализация)

## Задание: 3 графика + 3 вывода

Сделайте 3 графика на одном датасете:
1) динамика во времени,
2) сравнение категорий,
3) распределение.

К каждому графику напишите:
- что мы видим (1–2 предложения),
- что это значит (1–2 предложения),
- что делать (1 предложение).

## Критерии сдачи

- [ ] 3 графика
- [ ] У каждого есть подписи и заголовок
- [ ] Есть 3 мини-вывода (по одному на график)
