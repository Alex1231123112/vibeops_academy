# Промпты для AI: A0 (аналитический цикл)

База: [prompts-library/template.md](../../../prompts-library/template.md)

## 1) Превратить “хочу понять” в проверяемый вопрос

```
У меня цель/сомнение:
[вставь]

Сформулируй 3 проверяемых аналитических вопроса.
Для каждого:
- метрика (точное определение)
- сегмент
- период
- сравнение
- какие данные нужны
```

## 2) Найти риски интерпретации

```
Вот вывод/инсайт:
[вставь]

Перечисли 5 рисков интерпретации (сезонность, смещения, конфундеры) и как их проверить.
```
