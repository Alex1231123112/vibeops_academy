# Шпаргалка: аналитический вопрос (A0)

## Шаблон вопроса

```
Сегмент: [...]
Период: [...]
Метрика: [...]
Сравнение: с чем сравниваем (до/после, контроль/тест, сегмент A/B)
Критерий успеха: что будет “хорошо”
```

## Примеры

- “У новых пользователей (регистрация ≤ 7 дней) выросла ли конверсия в оплату после изменения онбординга по сравнению с прошлой неделей?”
- “У пользователей из канала X отличается ли retention D7 от канала Y за последний месяц?”
