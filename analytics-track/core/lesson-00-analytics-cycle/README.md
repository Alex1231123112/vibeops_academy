# A0: Аналитический цикл и постановка вопросов

## 🎯 Цели урока

- Понять цикл “вопрос → данные → обработка → метрики → вывод → решение”
- Научиться формулировать аналитический вопрос так, чтобы его можно было проверить
- Разделять **метрику**, **сегмент**, **сравнение** и **критерий успеха**

## ⏱️ Сколько времени

Ориентир: **45–90 минут**.

## 📖 Теория (минимум)

### Формула хорошего вопроса

“Для [сегмент] при [условие/период] изменилось ли [метрика] после [событие/изменение] по сравнению с [baseline]?”

### Типовые ловушки

- “Слишком общий вопрос” → нет критериев и сравнения
- “Нет определения метрики” → разные люди считают по-разному
- “Нет периода” → сезонность и случайность
