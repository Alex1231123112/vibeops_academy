# Шпаргалка: Linux/Терминал

## Навигация

| Команда | Описание |
|---------|----------|
| `pwd` | Текущая директория |
| `ls` | Список файлов |
| `ls -la` | Подробный список |
| `cd path` | Перейти |
| `cd ..` | На уровень вверх |

## Файлы

| Команда | Описание |
|---------|----------|
| `mkdir name` | Создать папку |
| `touch file` | Создать файл |
| `rm file` | Удалить файл |
| `rm -r dir` | Удалить папку |
