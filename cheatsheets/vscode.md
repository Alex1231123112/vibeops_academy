## Горячие клавиши

| Клавиши | Действие |
|---------|----------|
| Ctrl+S | Сохранить |
| Ctrl+P | Быстрый поиск файла |
| Ctrl+Shift+P | Command Palette |
| Ctrl+` | Терминал |
| Ctrl+B | Скрыть/показать сайдбар |
