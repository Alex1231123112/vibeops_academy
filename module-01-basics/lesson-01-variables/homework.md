# Домашнее задание: Урок 1.1

## Задание 1: Калькулятор с проверкой

Создайте калькулятор, который:
1. Просит ввести два числа
2. Просит выбрать операцию (+, -, *, /)
3. Выводит результат
4. Проверяет деление на ноль
5. Работает, пока пользователь не введет "выход"

**Промпт для AI:**
```
Напиши калькулятор на Python с бесконечным циклом. 
Пользователь вводит два числа и операцию. 
Программа выводит результат и спрашивает, хочет ли пользователь продолжить.
Добавь обработку ошибок (деление на ноль, некорректный ввод).
```

## Задание 2: TODO-лист

Создайте простой TODO-лист на списках.

**Требования:**
- Хранить задачи в списке
- Добавлять задачу
- Удалять задачу по номеру
- Показывать все задачи
- Отмечать задачу выполненной

**Промпт для AI:**
```
Напиши простой TODO-лист на Python. 
Используй список для хранения задач. 
Команды: добавить, удалить (по номеру), показать, отметить выполненной.
Интерфейс через input() в цикле.
```

## Критерии сдачи

- [ ] Код запускается без ошибок
- [ ] Обработаны типы (int, float для чисел)
- [ ] Есть проверка деления на ноль (для калькулятора)
- [ ] Код залит на GitHub
