# Промпты для AI: Урок 1.1 (Переменные)

## Базовый промпт

```
Напиши на Python программу, которая:
1. Спрашивает имя пользователя
2. Спрашивает возраст
3. Выводит "Привет, {имя}! Через 5 лет тебе будет {возраст + 5}"
Учти, что возраст нужно преобразовать в число (int).
```

## Продвинутый промпт

```
Создай программу для учета книг. Нужно:
- Хранить список книг (каждая книга — словарь с полями: название, автор, год)
- Добавлять новую книгу через input()
- Показывать все книги
- Искать книги по автору
Код должен быть разбит на функции.
```

## Промпт для отладки

```
У меня ошибка в коде:
[вставь код]

Ошибка: [вставь текст ошибки]

Проверь типы данных. input() возвращает строку — возможно, нужно int() или float().
```
