- Узнать про параметры, возвращаемые значения и type hints

## 📖 Теория

Функция — это переиспользуемый блок кода с именем. Вместо копирования одного и того же кода вы вызываете функцию.

```python
def greet(name: str) -> str:
    """Возвращает приветствие."""
    return f"Привет, {name}!"
```
