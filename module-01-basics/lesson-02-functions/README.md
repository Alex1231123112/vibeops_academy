## 📖 Теория

Функция — это переиспользуемый блок кода с именем. Вместо копирования одного и того же кода вы вызываете функцию.

```python
def greet(name: str) -> str:
    """Возвращает приветствие."""
    return f"Привет, {name}!"
```

### Как думать про функции (практично)

- **Сигнатура** — это “контракт”: что функция принимает и что возвращает.
- Хорошая функция делает **одну понятную вещь** и имеет **предсказуемый результат**.
- Если внутри много `if/else` и повторов — обычно стоит разбить на 2–3 функции.

### Типичные ошибки AI в функциях

- Возвращает не тот тип (например, `str` вместо `int`).
- Пишет функцию, но она **печатает** вместо того, чтобы **возвращать** значение.
- Путает область видимости: использует переменные, которых нет в аргументах.
