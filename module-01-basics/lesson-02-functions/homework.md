# Домашнее задание: Урок 1.2

## Задание: 3 функции с контрактами и примерами

Напишите **3 функции** с:
- type hints (аргументы и возвращаемое значение),
- docstring (1–3 строки: что делает функция, аргументы, возврат),
- примерами использования.

### Обязательные функции

1) `mean(a: float, b: float) -> float`  
Возвращает среднее арифметическое \( (a + b) / 2 \).

2) `is_palindrome(s: str) -> bool`  
Возвращает `True`, если строка — палиндром. Игнорируйте пробелы и регистр.

3) `parse_int(s: str) -> int | None`  
Возвращает число, если строка корректно парсится как целое (после `.strip()`), иначе `None`.

### Критерии сдачи

- [ ] У каждой функции есть type hints и docstring
- [ ] У каждой функции есть минимум 3 примера вызова (можно в `if __name__ == "__main__":` или отдельным блоком в README репозитория)
- [ ] `parse_int` не падает на плохих строках (например `"abc"`, `""`, `"  "`)
- [ ] Код залит на GitHub
