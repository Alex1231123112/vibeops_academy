# Промпты для AI: Урок 1.2 (Функции)

## Базовый промпт

```
Напиши функцию на Python, которая [опишите задачу].
Функция должна:
- Принимать параметры: [список]
- Возвращать: [что возвращает]
- Содержать docstring с описанием
- Использовать type hints
```

## 2) Разбей код на функции

```
Вот код, который сложно читать:
[вставь код]

Разбей его на 2–5 функций. Требования:
- у функций должны быть понятные имена
- минимум глобальных переменных
- добавь type hints
- поведение должно сохраниться
```

## 3) Найди контракт и граничные случаи

```
Я хочу функцию: [опиши идею].

1) Предложи сигнатуру (type hints) и что она должна возвращать.
2) Перечисли 5 граничных случаев.
3) Напиши реализацию и примеры вызова.
```

## 4) “Верни, а не печатай”

```
Вот функция от AI:
[код]

Сделай так, чтобы функция НЕ печатала результат, а возвращала его.
Если нужно, добавь отдельную функцию main(), которая печатает.
```
