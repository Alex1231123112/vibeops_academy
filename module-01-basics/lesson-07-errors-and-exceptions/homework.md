# Домашнее задание: Урок 1.7

## Задание: Устойчивый парсер конфигурации

Сделайте скрипт `read_config.py`, который:

1) просит путь к JSON-файлу (через `input()`),  
2) читает JSON,  
3) извлекает `{"host": "...", "port": 1234}`,  
4) печатает `host:port`.

### Требования к устойчивости

- Если файла нет — показать сообщение и попросить путь ещё раз
- Если JSON битый — показать сообщение и попросить путь ещё раз
- Если `port` не число или вне диапазона 1..65535 — сообщить и попросить путь ещё раз
- Ловить **конкретные** ошибки (`FileNotFoundError`, `json.JSONDecodeError`, `ValueError`, `KeyError`)

### Критерии сдачи

- [ ] Скрипт не падает на ошибках выше
- [ ] Есть 3 примера “плохих” входных данных и ожидаемое поведение (в README репозитория)
- [ ] Код залит на GitHub
