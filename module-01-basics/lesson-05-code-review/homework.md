# Домашнее задание: Урок 1.5

## Задание: Провести ревью кода

Проверьте код по чек-листу из урока (или из README).

### Что сделать

1. Возьмите код из предыдущего урока (1.4 — класс/задача) или любой свой небольшой скрипт (до 50–100 строк).
2. Пройдите по чек-листу:
   - Код запускается без ошибок?
   - Граничные случаи: пустой ввод, нулевые значения, пустые списки?
   - Имена переменных и функций понятные?
   - Нет ли дублирования, которое можно вынести в функцию?
   - Секреты и пароли не захардкожены?
3. Зафиксируйте результат: 3–5 пунктов «что проверил / что нашёл / что исправил».

### Формат сдачи

- Заметка в блокноте, комментарий в репо или пост в чат курса.
- Минимум: «Проверил [файл/функцию]. Нашёл: … Исправил: …»

### Критерии

- [ ] Использован чек-лист (из урока или свой)
- [ ] Перечислены хотя бы 2 проверки с результатом (ок / нашёл проблему / исправил)
- [ ] Код после правок (если были) в репо или приложен
