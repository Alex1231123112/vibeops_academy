# Промпты для AI: Урок 1.8 (Файлы/JSON/typing)

## 1) Сохраняй данные в JSON

```
Напиши программу на Python, которая хранит задачи в файле data.json.

Функции:
- load_tasks(path) -> list[dict]
- save_tasks(path, tasks) -> None

Требования:
- JSON должен быть читаемым (indent=2)
- Кириллица не должна экранироваться (ensure_ascii=False)
- Если файла нет — возвращай пустой список
```
