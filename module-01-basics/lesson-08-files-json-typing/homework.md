# Домашнее задание: Урок 1.8

## Задание: TODO с сохранением в JSON

Возьмите TODO-менеджер из урока 1.6 и добавьте сохранение задач в JSON-файл.

### Требования

- Файл: `data.json` (в корне проекта)
- При запуске программа загружает задачи из `data.json` (если файла нет — пустой список)
- После любой команды, которая меняет задачи (`add`, `done`, `delete`), программа сохраняет обновлённый список
- JSON пишем с:
  - `ensure_ascii=False`
  - `indent=2`
- Используйте type hints (минимум для функций чтения/записи)

### Критерии сдачи

- [ ] Задачи сохраняются между запусками
- [ ] На пустом/отсутствующем файле программа не падает
- [ ] Есть функции `load_tasks()` и `save_tasks()`
- [ ] Код залит на GitHub
