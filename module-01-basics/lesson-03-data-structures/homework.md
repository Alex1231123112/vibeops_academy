# Домашнее задание: Урок 1.3

## Задание: Хранение и поиск данных

Реализуйте небольшой скрипт, который использует **list**, **dict** и при желании **set**.

### Вариант A: Каталог товаров

- Храните список товаров: каждый товар — словарь с полями `name`, `price`, `category`.
- Функция поиска по категории: возвращает список товаров заданной категории.
- Функция «уникальные категории»: возвращает множество категорий (без дубликатов).

### Вариант B: Счётчик слов

- Прочитайте текст (из переменной или файла).
- Подсчитайте частоту каждого слова в словаре: `dict[str, int]`.
- Выведите топ-5 самых частых слов.

### Критерии сдачи

- [ ] Используются и list, и dict (и при желании set)
- [ ] Код не падает на пустом вводе / пустом списке
- [ ] Есть хотя бы одна функция с понятной сигнатурой
- [ ] Код залит в репо или приложен к отчёту
