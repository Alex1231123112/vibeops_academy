# Промпты для AI: Урок 1.4 (ООП)

## 1) Простой класс

```
Напиши на Python класс Task (задача) с полями:
- title: str
- done: bool (по умолчанию False)

Методы:
- toggle() — переключает done (True <-> False)
- __str__() — возвращает строку вида "[x] Название" или "[ ] Название"

Без наследования, без лишних библиотек.
```

## 2) Класс с валидацией

```
Напиши класс User с полями email и name.
В __init__ проверяй, что email содержит символ @. Если нет — выброси ValueError с понятным сообщением.
Добавь метод __repr__ для удобного вывода при отладке.
```
