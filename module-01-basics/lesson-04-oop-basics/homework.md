# Домашнее задание: Урок 1.4

## Задание: Прочитать и починить ООП-код

Выберите один из вариантов.

### Вариант A: Код от AI

1. Попросите AI сгенерировать класс на Python (например: «Класс Task с полями title, done, методом toggle»).
2. Запустите код, найдите ошибку (например, опечатка в атрибуте, забытый `self`).
3. Исправьте и закоммитьте изменения. В PR или в комментарии опишите: что было сломано и как починили.

### Вариант B: Готовый репозиторий

1. Возьмите любой небольшой учебный проект с классами (свой старый или пример из курса).
2. Прочитайте один класс: что хранит (атрибуты), что умеет (методы).
3. Добавьте один новый метод или атрибут, не ломая существующее поведение. Сделайте коммит.

### Критерии сдачи

- [ ] Код с классом(ами) запускается без ошибок
- [ ] Есть краткое описание: что делает класс, что вы изменили или починили
- [ ] Изменения в Git (репо или патч)
