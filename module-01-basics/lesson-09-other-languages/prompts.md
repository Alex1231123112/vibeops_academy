# Промпты для AI: Урок 1.9 (Другие языки)

## 1) Перевести скрипт на Go

```
У меня есть такой скрипт на Python:

[вставь код, например: чтение JSON-файла и вывод полей]

Переведи его на Go. Сохрани ту же логику: чтение файла, разбор JSON, вывод. Добавь обработку ошибок в стиле Go (if err != nil).
```

## 2) Перевести фрагмент с Python на TypeScript

```
Вот функция на Python:

[вставь код, например: def build_url(host: str, port: int) -> str: ...]

Напиши эквивалент на TypeScript: типизированные аргументы и возврат, тот же контракт функции.
```

```

```
