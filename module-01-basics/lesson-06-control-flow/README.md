# Урок 1.6: Управляющие конструкции (control flow)

После переменных, функций и структур данных вам нужен практический минимум: **условия, циклы, “ранний выход” и простая CLI-логика**. Это основа почти любого скрипта и сервиса, а также место, где AI часто допускает ошибки (бесконечные циклы, неверные условия, отсутствие валидации).

## 🎯 Цели урока

- Условия: `if / elif / else`
- Циклы: `for`, `while`, `break`, `continue`
- `range()`, перечисление `enumerate()`
- Понимать “guard clauses” (ранние проверки и выход)
- Научиться строить простой CLI-цикл “меню команд”

## ⏱️ Сколько времени

Ориентир: **2–3 часа** (прочитать + собрать CLI TODO Manager).

## 📖 Теория и примеры

### 1) Условия

```python
age = 18
if age < 0:
    print("Возраст не может быть отрицательным")
elif age < 18:
    print("Доступ запрещён")
else:
    print("Доступ разрешён")
```

**Типичная ошибка AI:** условие “наоборот” или пропуск проверок на пустую строку/`None`.

### 2) Циклы и “ранний выход”

```python
while True:
    cmd = input("Команда (add/list/exit): ").strip().lower()
    if cmd == "exit":
        break
    if cmd == "":
        continue
    print("Вы ввели:", cmd)
```

### 3) Обход списка с номерами

```python
tasks = ["вымыть посуду", "прочитать урок", "сделать ДЗ"]
for i, task in enumerate(tasks, start=1):
    print(i, task)
```

### 4) (Опционально) `match/case` как “switch”

`match/case` доступен в Python 3.10+:

```python
cmd = "list"
match cmd:
    case "add":
        print("Добавляем…")
    case "list":
        print("Показываем…")
    case _:
        print("Неизвестная команда")
```
