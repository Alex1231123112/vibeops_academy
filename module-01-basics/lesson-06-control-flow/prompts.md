# Промпты для AI: Урок 1.6 (Control flow)

## 1) CLI-меню с командами

```
Напиши CLI-программу на Python с бесконечным циклом и командами:
- add (добавить задачу)
- list (показать задачи с номерами)
- done (отметить задачу выполненной по номеру)
- delete (удалить задачу по номеру)
- exit (выйти)

Требования:
- Ввод команды и аргументов через input()
- Валидация: пустая команда, неверный номер, номер вне диапазона
- Используй функции (add_task, list_tasks, mark_done, delete_task)
- Храни задачи в list[dict] с полями: text (str), done (bool)
```
