# Домашнее задание: Урок 1.6

## Задание: CLI TODO Manager

Сделайте программу “TODO менеджер” в терминале.

### Требования

- Команды: `add`, `list`, `done`, `delete`, `exit`
- Список задач: `list[dict]`, где каждая задача имеет поля:
  - `text: str`
  - `done: bool`
- Валидация ввода:
  - пустые команды
  - неверный номер задачи
  - номер вне диапазона

### Критерии сдачи

- [ ] Программа не падает на плохом вводе
- [ ] Есть функции (не один огромный `while True`)
- [ ] Команда `list` показывает задачи с номерами
- [ ] Есть пример использования (несколько строк “как пользоваться”) в README репозитория
- [ ] Код залит на GitHub
