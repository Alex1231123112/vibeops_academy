# Промпты для начинающих

## Генерация кода

```
Напиши на Python программу, которая [задача].
Учти: input() возвращает строку — используй int() или float() где нужно.
Добавь комментарии на русском.
```

## Объяснение кода

```
Объясни, что делает этот код, построчно:
[код]
Простыми словами, для новичка.
```

## Исправление ошибки

```
У меня ошибка: [текст ошибки]
Вот код: [код]
Объясни причину и исправь.
```
