# Универсальный шаблон промпта (для задач по разработке)

Используйте этот шаблон, когда хотите получить стабильный и проверяемый результат от AI.

## Шаблон (копируй-вставляй)

```
Контекст:
- проект/файл: ...
- стек/версии: ...
- где точка входа: ...

Цель:
- нужно: ...

Ограничения:
- нельзя: ...
- нужно сохранить поведение: да/нет
- зависимости: (можно/нельзя добавлять)

Формат ответа:
- дай: (патч / список шагов / код + тесты)

Критерии качества:
- edge cases: ...
- безопасность/секреты: (ничего не хардкодить)
- тест-план: (как проверить)
```

## Мини-набор усилителей (по ситуации)

- “Покажи **патч**, а не пересказ.”
- “Не меняй поведение, только структуру/читаемость.”
- “Добавь 3–5 edge cases и как их проверить.”
- “Сначала задай до 7 уточняющих вопросов.”
