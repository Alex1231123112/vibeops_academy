# Промпты для отладки

## Анализ ошибки

```
Ошибка: [полный traceback]
Код: [релевантный фрагмент]
Что не так? Как исправить?
```

## Поиск бага

```
Программа должна [ожидаемое поведение], но делает [фактическое].
Код: [код]
Помоги найти причину.
```

## Логирование

```
Добавь логирование в этот код для отладки:
[код]
Логи должны показывать значения ключевых переменных.
```
