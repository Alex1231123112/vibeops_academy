# Промпты для архитектуры

## Проектирование

```
Спроектируй структуру приложения для [описание].
Нужно: модули, основные классы/функции, API endpoints.
Опиши в виде списка и кратких описаний.
```

## Рефакторинг

```
Отрефактори этот код:
[код]
Сохрани поведение. Улучши читаемость и структуру.
Разбей на функции/модули если нужно.
```

## API дизайн

```
Спроектируй REST API для [сущность].
Нужны: endpoints, методы, форматы запроса/ответа.
В формате OpenAPI или списком.
```
