# Промпты для Docker

## С agency-agents

Установите [agency-agents](https://github.com/msitarzewski/agency-agents). Для безопасности — **Security Engineer**, для пайплайнов — **DevOps Automator**.

## Dockerfile

```
Создай Dockerfile для Python-приложения.
- Python 3.12
- Зависимости из requirements.txt
- Рабочая директория /app
- CMD для запуска main.py
Оптимизируй для размера образа.
```

## Docker Compose

```
Создай docker-compose.yml:
- Сервис app (сборка из текущей папки)
- Сервис postgres
- app зависит от db
- Переменные для подключения к БД
```
