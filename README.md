## 📚 О курсе

**VibeOps** — это бесплатный курс для тех, кто хочет научиться программировать с помощью AI, но не стать заложником нейросетей. Вы пройдете путь от первой переменной до продакшн-проекта в облаке.

### 🎯 Для кого
- Новички, которые хотят программировать с AI
- Разработчики, желающие освоить DevOps
- Продуктовые менеджеры и аналитики
- Все, кто устал от "код не работает, AI не помогает"

### 📋 Что вы изучите
- ✅ Основы Python (достаточно, чтобы понимать код от AI)
- ✅ Git и терминал — must have для любого разработчика
- ✅ Docker — упаковка приложений в контейнеры
- ✅ CI/CD — автоматический деплой при каждом пуше
- ✅ Облачные платформы (Render, Railway)
- ✅ Промпт-инжиниринг для программирования
- ✅ Архитектуру приложений

## 🛠 Инструменты

### Админка для канала (на базе [HT](https://github.com/Alex1231123112/HT))

Полнофункциональная админ-панель: контент-план, рассылки, аналитика.

```bash
cd tools
git clone https://github.com/Alex1231123112/HT.git ht-admin
cd ht-admin
cp ../ht-admin-vibeops/.env.vibeops.example .env
# Отредактируйте .env: BOT_TOKEN, JWT_SECRET, ADMIN_DEFAULT_PASSWORD

docker compose -f docker-compose.yml -f ../docker-compose.vibeops.yml up -d --build
```

Админка: **http://localhost:8800**

### Простая админка (без БД)

[telegram-admin](tools/telegram-admin/) — одна форма для быстрой публикации поста.
